# ☀️ YWS Energia Solar

Sistema web para simulação de energia solar fotovoltaica.

## 📋 Sobre o Projeto

A YWS Energia Solar oferece um simulador online que permite calcular rapidamente:

- Economia mensal estimada
- Economia anual
- Quantidade de placas solares necessárias
- Potência do sistema (kWp)
- Investimento estimado
- Tempo de retorno do investimento (Payback)

## 🚀 Funcionalidades

✅ Simulador fotovoltaico online

✅ Cálculo automático de consumo energético

✅ Dimensionamento de placas solares

✅ Estimativa de investimento

✅ Cálculo de retorno financeiro

✅ Integração com WhatsApp

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Service Worker (PWA)
- Vercel
